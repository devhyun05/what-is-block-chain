# what-is-block-chain
A web application to explain blockchain concept


## Tech Stack

**Client:** HTML5, CSS3, JavaScript, React



## Run Locally


Install dependencies

```bash
  cd client && npm install
```

Start the frontend 

```bash
  cd client
  npm start
```
